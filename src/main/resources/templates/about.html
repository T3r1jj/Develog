About 

    Develog is a web application inspired by James Routley's
idea of using a logbook to improve your programming. The app allows
users to create one private and tagged note per day. Simple Markdown together with PlantUML speeds up and facilitates
the process of describing a solution to a problem. The app is open source - see the repository on Github.
